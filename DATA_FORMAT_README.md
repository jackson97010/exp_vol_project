# 台股一分鐘K線成交量資料格式文檔

## 資料概述
此資料集包含台灣股市每日各股票的一分鐘K線成交量相關數據。

## 檔案資訊

### 檔案位置
- **主要目錄**：`D:\03_預估量相關資量\tw_kbar_1m_vol_exp\`
- **檔案格式**：Parquet (高效的列式儲存格式)
- **檔案命名規則**：`vol_exp_YYYYMMDD.parquet`
  - 範例：`vol_exp_20250901.parquet` 表示 2025年9月1日的資料

### 資料結構

#### 索引 (Index)
- **名稱**：`datetime` (原始名稱為 `k_time`，已修正)
- **類型**：`datetime64[us]` (微秒精度的日期時間)
- **格式**：`YYYY-MM-DD HH:MM:SS`
- **範例**：`2025-09-01 09:00:00`, `2025-09-01 09:01:00`
- **時間範圍**：每個交易日 09:00 至 13:30
- **頻率**：每分鐘一筆資料
- **每日資料筆數**：266筆 (從 09:00 到 13:30 的每分鐘)

#### 欄位 (Columns)
- **欄位名稱**：股票代碼
- **格式**：字串型態 (例如：'1101', '1102', '2330' 等)
- **數量**：依日期不同，約 500-600 支股票
- **內容**：各股票在該時間點的成交量相關數值

### 資料範例

```

2025-09-01 09:00:00     XXX     XXX     XXX     XXX     XXX

2025-09-01 09:02:00     XXX     XXX     XXX     XXX     XXX
...
2025-09-01 13:30:00     XXX     XXX     XXX     XXX     XXX
```

### 資料特徵
- **資料型態**：數值型資料（推測為成交量或成交量相關指標）
- **更新頻率**：每日更新
- **歷史資料**：包含多個交易日的歷史資料

## 使用範例

### 讀取資料
```python
import pandas as pd

# 讀取特定日期的資料
df = pd.read_parquet(r'D:\03_預估量相關資量\tw_kbar_1m_vol_exp\vol_exp_20250901.parquet')
```

### 查詢特定股票在特定時間的數值
```python
# 查詢 2330(台積電) 在 2025-09-01 09:30 的數值
value = df.loc['2025-09-01 09:30:00', '2330']
```

### 查詢特定股票的全日資料
```python
# 取得 2330 的全日資料
stock_data = df['2330']
```

## 注意事項

1. **索引名稱歷史變更**：
   - 原始索引名稱為 `k_time`（格式：'0930'）
   - 已轉換為 `datetime` 格式並更名為 `datetime`

2. **備份檔案**：
   - 轉換過程中可能產生的備份檔案：
     - `*_backup.parquet` - 資料轉換備份
     - `*_name_backup.parquet` - 索引名稱修正備份
     - `*_index_backup.parquet` - 索引修正備份

3. **資料完整性**：
   - 每個交易日應包含 266 筆時間點資料
   - 股票數量可能因日期不同而有所變化
